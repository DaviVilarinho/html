Este Site!

Bom pode parecer simples, mas esse site me fez passar por muitas situações divertidas. Configuração do DNS, uso de Containers para manutenção do site, instância EC-2, Lightsail, Reverse-Proxy com Nginx, criptografia SSL, Certbot com Docker e sem, scripts de automação... Ufa! Deve ter mais coisa, mas o trabalho foi muito e a memória foi pouca!

Sigilo

No momento, ainda não foi efetivada, mas consegui uma reestruturação completa com containers, utilizando Docker-Compose, do software principal, seu banco de dados (que migrei com sucesso) e Certbot.
